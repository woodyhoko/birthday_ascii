# Birthday ASCII Art Generator

A browser-based tool that **generates custom ASCII art from any image** and lets you print it as a personalized birthday greeting.

**[▶ Try It Live](https://woodyhoko.github.io/birthday_ascii)**

---

## How It Works

1. Click **"find a cake"** — fetches a random cake image from the web
2. Click **"bake your cake"** — converts the image to ASCII art on an HTML5 Canvas using a configurable character set
3. Click **"take home"** — downloads your ASCII art
4. Don't want a cake? Type anything in the **"I want a ____"** field and get custom ASCII art of it instead

---

## Features

- 🎂 Auto-fetches random birthday cake images
- 🔤 Configurable character set for the ASCII palette (dense `#KDGLftji+;,:.` to sparse)
- 🎨 Canvas-rendered ASCII art
- 💾 One-click download

---

## Stack

- Vanilla JavaScript + jQuery
- HTML5 Canvas for pixel-to-character conversion
